Okaerii 🍵

Aplikasi macOS menu bar untuk membangun suasana fokus, relaksasi, dan tidur dengan soundscape ber‑kualitas tinggi. Sepenuhnya offline dan menjaga privasi.

Download untuk macOS Lihat di GitHub Versi Terbaru Dukung di Patreon
Audio ber‑kualitas tinggi disertakan langsung dalam aplikasi untuk pengalaman terbaik — tanpa streaming, buffering, atau langganan.

Cara Install

1. Download

Unduh Okaerii.dmg dari halaman rilis.

2. Drag & Drop

Buka DMG, seret Okaerii ke folder Applications.

3. Mulai

Klik ikon 🍵 di menu bar untuk mulai mendengarkan.
