Okaerii 🍵

A minimalist, open-source macOS menu bar app that crafts high‑fidelity ambient soundscapes to help you focus, relax, or sleep — entirely offline and private.

Features

  • High‑Fidelity Audio: Bundled studio‑grade soundscapes for a rich, immersive experience.
  • 100% Offline & Private: No internet, no buffering, no tracking, no subscriptions.
  • Menu Bar Native: Lightweight, always available via a simple 🍵 icon.
  • Curated Scenes: Handcrafted environments like Deep Work, Midnight Rain, Forest Zen.
  • Custom Mixes: Blend layers and fine‑tune individual volumes to taste.
  • Gapless Looping: Seamless, infinite playback powered by a custom audio engine.
  • Minimalist UI: Clean, distraction‑free interface built with SwiftUI.
